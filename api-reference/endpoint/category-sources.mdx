---
title: 'Get category sources'
description: 'Retrieve all sources collected for a given data category in chronological order, up to 90 days back.'
api: 'GET https://parsagon.io/api/events/v1/categories/{category_id}/sources/'
---

Returns a chronological list of all sources collected for the specified category. Results cover up to the last 90 days.

## Path parameters

<ParamField path="category_id" type="number" required>
  The ID of the data category to retrieve sources for.
</ParamField>

## Response

<ResponseField name="sources" type="array">
  Array of source objects in chronological order (oldest first).

  <Expandable title="Source fields">
    <ResponseField name="url" type="string">
      URL of the source document.
    </ResponseField>
    <ResponseField name="title" type="string">
      Title of the document.
    </ResponseField>
    <ResponseField name="date" type="string">
      Date when Parsagon collected the document (ISO 8601, UTC).
    </ResponseField>
    <ResponseField name="text" type="string">
      Full text (or main body) of the document.
    </ResponseField>
    <ResponseField name="summary" type="string">
      AI-generated 1–2 sentence summary of the document.
    </ResponseField>
    <ResponseField name="type" type="string">
      Type of document. Either `"ARTICLE"` (press releases, announcements, generic articles, and similar content) or `"LAW"` (bills and legislation).
    </ResponseField>
    <ResponseField name="extra_data" type="object">
      For `"LAW"` documents, an object containing additional legislative metadata. May include `status`, `last_action`, and `last_action_date` — not all fields are present on every document. For non-`"LAW"` documents, this is usually empty.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="error" type="string | null">
  Error message if an error occurred; otherwise `null`.
</ResponseField>

## Example (cURL)

```bash
curl -X GET "https://parsagon.io/api/events/v1/categories/1/sources/" \
  -H "Authorization: Token YOUR_API_KEY_HERE"
```
