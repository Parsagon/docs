---
title: 'Get category sources'
description: 'Retrieve all sources collected for a given data category in chronological order, up to 90 days back.'
api: 'GET https://parsagon.io/api/events/v1/categories/{category_id}/sources/'
---

Returns a chronological list of all sources collected for the specified category. Results cover up to the last 90 days.

## Path parameters

<ParamField path="category_id" type="number" required>
  The ID of the data category to retrieve sources for.
</ParamField>

## Response

<ResponseField name="count" type="number">
  Total number of sources available for the category.
</ResponseField>

<ResponseField name="next" type="string | null">
  URL for the next page of results, or `null` if this is the last page.
</ResponseField>

<ResponseField name="previous" type="string | null">
  URL for the previous page of results, or `null` if this is the first page.
</ResponseField>

<ResponseField name="results" type="array">
  Array of source objects in chronological order (oldest first). Each object has the same fields as a [Quick Search](/api-reference/endpoint/quick-search) result.
</ResponseField>
